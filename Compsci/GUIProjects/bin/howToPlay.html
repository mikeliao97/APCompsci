How to Play

Overview
Minesweeper is a game of logic and probability. The game begins with a grid of cells some of which contain hidden mines. The goal of the game is to reveal all the cells that are safe by clicking on them. A cell is safe it it doesn't contain a mine. Once the user has revealed all safe cells, the game is won

Type of Cells
1. The game begins with unrevealed cells

If you click an unrevealed cell you may uncover a number cell. Number cells range from 1 to 8 and represent the number of neighboring cells containing mines. For example, in the game below four cells are touching 1 neighboring mine and three cells are touching 2 neighboring mines
